## Intro

Project ini menggunakan Google Apps Script 

### Tutorial 
Clone this project

```bash
> git clone https://github.com/ArugaZ/whatsapp-bot.git
> cd whatsapp-bot
```

Install the dependencies:

```bash
> npm install 
> npm install gify-cli -g
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```

after running it you need to scan the qr
